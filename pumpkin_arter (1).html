A desk for OpenAI's image model. Attach nothing for a plain text-to-image render, or attach references and just describe what to do with them — combine them, copy a style, turn them into a thumbnail. Your API key stays in this tab only.

Used only for this session, held in page memory. Get a key at platform.openai.com/api-keys. GPT Image models may require organization verification on your OpenAI account first.
◐ Drop one photo here, or Its look — palette, lighting, texture — gets carried over into the render. Swap it any time by dropping a new one.

Images are sent to the model in the numbered order shown below — just describe what you want done with them (combine, restyle, extract a look, turn into a thumbnail, etc.). No reference images attached means a straight text-to-image render. If a style photo is attached, its look is applied automatically — no need to describe it.
GPT Image only renders three real shapes (1:1, 3:2 landscape, 2:3 portrait) — presets above snap to the closest one. Ultra-wide banners will need a manual crop afterward.
◐ Nothing rendered yet — write a prompt (and attach references if you want) and press generate.
Calls api.openai.com directly from your browser using your key — no images attached uses /v1/images/generations, one or more attached uses /v1/images/edits with all of them plus your prompt. Heads up: OpenAI's API does not reliably support direct browser calls (no CORS headers), so this may fail with a network error even with a valid key — if so, you'd need a small backend relay to forward the request server-side.
